SOLID es un conjunto de 5 principios de diseño en la programación orientada a objetos. Su objetivo es ayudarte a escribir código que sea:

📦 Fácil de mantener

🧩 Reutilizable

🛠️ Escalable

🔄 Fácil de testear y modificar sin romper otras partes del sistema
